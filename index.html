JavaScript

JavaScript (JS) is a lightweight, interpreted, programming language with first-class functions. Most well-known as the scripting language for Web pages.

Assignment Operators

Changes the value of the variable on the left of the operator.

Comparison Operators

Does not change any values, but returns a boolean (true or false) depending on whether the statement evaluates as true or false.

Arithmeic Operators

Arithmetic operators take numerical values (either literals or variables) as their operands and return a single numerical value. The standard arithmetic operators are addition (+), subtraction (-), multiplication (*), and division (/).

Variables

Variables can be thought of as containers used to store information. They allow for a way to label data with a descriptive name.

Variable naming conventions

Lower camel case: Use lowerCamelCase when naming JavaScript variables. Start with a lowercase letter, and if the variable name is more than one word, remove all spaces and capitalize the first letter of each subsequent word.

Functions

A function is a block of code that performs an action and returns a result; optionally takes arguments.

Methods

A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Arguments

Some methods take one or more arguments that provide the method with additional information to help it perform its action.

Parameters

A parameter is a named variable passed into a function. Parameter variables are used to import arguments into functions.

Return

The return value is the method's response.

Chaining Methods

Method chaining is a technique that can be used to simplify code in scenarios that involve calling multiple functions on the same object consecutively.

Strings

A JavaScript data type that represents the exact text of whatever is enclosed in the quotes

Booleans

true and false are booleans. They are not strings - they simply represent being true or false.

Undefined

Undefined is a primitive and is the sole value of the Undefined type. Any property that has not been assigned a value, assumes the undefined value. A function without a return statement, or a function with an empty return statement returns undefined. The value of an unsupplied function argument is undefined.

Nan

NaN stands for 'not a number'. NaN is a type of number.

Alert

alert() opens a dialog box and returns undefined.

Comments

anything following // on a line in JavaScript is ignored by the computer. Comments are a convenient way to leave notes in your code for yourself or other programmers.

JQuery

jQuery is a cross-platform JavaScript library designed to simplify the client-side scripting of HTML.

Attributes

Attributes provide additional information about HTML elements. They are usually put after an html tag name,
e.g. '< img src="w3schools.jpg" width="104" height="142" >.' the src, width, and height are all attributes
